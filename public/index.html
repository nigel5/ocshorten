Ocurl

Ocurl (prononced oh-curl), the URL shortener REST API

Example

Get a shortened url for https://github.com

fetch('https://ocurl.io/api/v1/url?q=https://github.com')
  .then((res) => res.json())
  .then((json) => console.log(json));
// Output:
// {
//   "data": {
//     "url": "ocurl.io/ou6p1"
//   }
// }

Resources

There are only two possible response types from the service. Data or error. 

// Data response
{
  "data": Object
}

// Error response
{
  "err": Object,
  "message": string
}

Available endpoints

Get a short URL

GET /api/v1/url
            Query Parameters
              - q {string} The destination URL

Decode a short url or path

GET /api/v1/decode
            Query Parameters
              - q {string} The short URL or path

System status check

GET /api/v1/health
            Query Parameters
              none
